// tag::EN[]
==== Product

Product data is data whose values have several, fixed attributes.
Also known as *records*, *structs*, *tuples* or *and data*.

Example:

An *address* has the following attributes:

- name, and
- street, and
- street number, and
- city, and
- zip code

// end::EN[]
